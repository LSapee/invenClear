# InvenClear

인벤 게시판의 **내 글 / 내 댓글** 목록에서 여러 건을 선택하여 한 번에 삭제할 수 있는 Chrome 확장 프로그램입니다.

## 주요 기능

- **내 글 삭제** (`?my=post`) — 체크박스로 여러 글을 선택하고 한 번에 삭제
- **내 댓글 삭제** (`?my=opi`) — 각 글의 '댓글 보기'를 눌러 내 댓글을 확인하고 선택 삭제

## 설치

### Chrome 웹스토어
> 준비 중

### 로컬 설치 (개발자 모드)
1. 이 저장소를 클론 또는 다운로드
2. Chrome → `chrome://extensions/` 접속
3. 오른쪽 위 **개발자 모드** ON
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. `src/` 폴더 선택

## 사용법

1. 인벤에 로그인
2. 내 글 목록(`?my=post`) 또는 내 댓글 목록(`?my=opi`) 페이지 접속
3. 삭제할 항목을 체크박스로 선택
4. **선택 삭제** 버튼 클릭

## 프로젝트 구조

```
invenClear/
├── src/
│   ├── manifest.json   # MV3 매니페스트
│   ├── content.js      # 체크박스 주입 + 삭제 로직
│   ├── styles.css      # UI 스타일
│   └── img/            # 아이콘 및 이미지
└── PRIVACY.md          # 개인정보처리방침
```

## 주의사항

- 삭제된 글/댓글은 **복구할 수 없습니다.** 신중히 사용해 주세요.
- 이 확장 프로그램은 인벤 공식 제품이 아니며, 인벤과 어떠한 제휴 관계도 없습니다.
