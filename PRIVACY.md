# 개인정보처리방침 (Privacy Policy)

**Inven Clear**

최종 수정일: 2026-04-21

## 수집하는 개인정보

본 확장 프로그램(Inven Clear)은 사용자의 개인정보를 외부 서버로 수집, 저장, 전송하지 않습니다.

- 수집하는 개인정보: 없음
- 외부 서버로 전송하는 데이터: 없음
- 분석(Analytics) 도구 사용: 없음

## 로컬 저장소 사용

본 확장 프로그램은 기능 동작을 위해 브라우저 저장소를 사용합니다.

- `chrome.storage.sync`: No인장 가리기 설정값 저장
- `sessionStorage`: 10추글 제외 기능을 위해 현재 브라우저 세션에서 추천 10 이상 게시글 ID 임시 저장

저장되는 값은 확장 프로그램 기능 설정 또는 임시 상태값이며, 인벤 계정 정보, 비밀번호, 쿠키, 게시글/댓글 내용은 저장하지 않습니다.

`chrome.storage.sync`에 저장된 설정값은 Chrome 동기화가 켜져 있는 경우 사용자의 Chrome 계정 환경에 따라 다른 브라우저와 동기화될 수 있습니다.

## 동작 방식

본 확장 프로그램은 사용자가 인벤(inven.co.kr)에 직접 로그인한 세션을 통해서만 해당 사이트의 삭제 및 출석체크 기능을 호출합니다. 이 과정에서 어떠한 데이터도 제3자에게 수집되거나 전달되지 않습니다.

## 권한

본 확장 프로그램은 다음 권한을 사용합니다.

- `https://www.inven.co.kr/*`: 인벤 게시판의 내 글/내 댓글 정리 및 No인장 가리기 기능
- `https://imart.inven.co.kr/*`: 인벤 아이마트 출석체크 페이지 열기 및 도장 찍기 기능
- `storage`: 확장 프로그램 설정값 저장
- `windows`: 출석체크용 작은 popup window 열기
