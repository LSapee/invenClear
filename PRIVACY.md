# 개인정보처리방침 (Privacy Policy)

**Inven Clear**

최종 수정일: 2026-06-10

## 수집하는 개인정보

본 확장 프로그램(Inven Clear)은 사용자의 개인정보를 개발자 서버 또는 제3자 분석 서버로 수집, 저장, 전송하지 않습니다.

- 수집하는 개인정보: 없음
- 개발자 서버로 전송하는 데이터: 없음
- 분석(Analytics) 도구 사용: 없음

## 로컬 저장소 사용

본 확장 프로그램은 기능 동작을 위해 브라우저 저장소를 사용합니다.

- `chrome.storage.sync` 또는 `chrome.storage.local`: No인장 가리기 및 전투력 표시 설정값 저장
- `sessionStorage`: 10추글 제외 기능을 위해 현재 브라우저 세션에서 추천 10 이상 게시글 ID 임시 저장

저장되는 값은 확장 프로그램 기능 설정 또는 임시 상태값이며, 인벤 계정 정보, 비밀번호, 쿠키, 게시글/댓글 내용은 저장하지 않습니다.

`chrome.storage.sync`에 저장된 설정값은 Chrome 동기화가 켜져 있는 경우 사용자의 Chrome 계정 환경에 따라 다른 브라우저와 동기화될 수 있습니다.

## 동작 방식

본 확장 프로그램은 사용자가 인벤(inven.co.kr)에 직접 로그인한 세션을 통해서만 해당 사이트의 삭제 기능을 호출합니다. No인장 가리기와 주사위 찾기 기능은 사용자의 브라우저 안에서 현재 페이지의 게시글/댓글 DOM을 읽어 화면 표시와 검색 결과를 계산합니다.

전투력 표시 기능은 인벤 게시글/댓글의 닉네임을 기준으로 인벤 회원 인벤토리 페이지 및 인벤 게임 프로필 API를 조회해 메이플 전투력을 화면에 표시합니다. 이 요청은 전투력 표시 기능이 켜져 있을 때 인벤 도메인(`www.inven.co.kr`)으로만 전송되며, 조회 대상 닉네임과 사용자의 인벤 로그인 세션 쿠키가 브라우저 정책에 따라 함께 사용될 수 있습니다.

이 과정에서 확장 프로그램 개발자가 별도로 운영하는 서버로 데이터가 수집되거나 전달되지 않습니다.

## 권한

본 확장 프로그램은 다음 권한을 사용합니다.

- `https://www.inven.co.kr/*`: 인벤 게시판의 내 글/내 댓글 정리, No인장 가리기, 주사위 찾기, 전투력 조회 기능
- `https://m.inven.co.kr/*`: 모바일 인벤 게시판의 No인장 가리기, 전투력 표시, 추천/비추천 표시 기능
- `https://maple.inven.co.kr/*`: 메이플 인벤 게시판 기능 지원
- `storage`: 확장 프로그램 설정값 저장
