# 개인정보처리방침 (Privacy Policy)

**Inven Clear**

최종 수정일: 2026-04-15

## 수집하는 개인정보

본 확장 프로그램(Inven Clear)은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

- 수집하는 개인정보: 없음
- 외부 서버로 전송하는 데이터: 없음
- 쿠키 및 로컬 저장소 사용: 없음
- 분석(Analytics) 도구 사용: 없음

## 동작 방식

본 확장 프로그램은 사용자가 인벤(inven.co.kr)에 직접 로그인한 세션을 통해서만 해당 사이트의 삭제 기능을 호출하며, 이 과정에서 어떠한 데이터도 제3자에게 수집되거나 전달되지 않습니다.

## 권한

본 확장 프로그램은 `https://www.inven.co.kr/*` 호스트 권한만 사용하며, 인벤 게시판 페이지에서만 동작합니다.
